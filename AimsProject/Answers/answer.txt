1. Is JAVA a Pass by Value or a Pass by Reference programming language?
JAVA is a Pass by Value programming language. This means that when you pass an argument to a method, you are passing a copy of the value. If the argument is a  type like int, double, etc., the method receives a copy of the value. If the argument is an object reference, the method receives a copy of the reference, which points to the same object. 
JAVA là một ngôn ngữ lập trình Pass by Value. Điều này có nghĩa là khi truyền một đối số vào một phương thức, ta đang truyền một bản sao của giá trị đó. Nếu đối số là một kiểu như int, double, v.v., phương thức sẽ nhận được một bản sao của giá trị. Nếu đối số là một tham chiếu đối tượng, phương thức sẽ nhận được một bản sao của tham chiếu, trỏ đến cùng một đối tượng.
